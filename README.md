# Pong

This is my attempt to recreate the classic arcade game [Pong](https://www.youtube.com/watch?v=fiShX2pTz9A) using C++ and SDL2.

### Controls

W: Move paddle up  
S: Move paddle down  

#### Note
The ball gets a slight speed boost when it hits the edge of a paddle.
